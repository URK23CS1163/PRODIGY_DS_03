# PRODIGY_DS_03

## 🌳 Task 03 - Decision Tree Classifier

### 📌 Task Description:
Build a Decision Tree Classifier to predict whether a customer will purchase a product or service based on their demographic and behavioral data.

The model is trained using the Bank Marketing dataset from the UCI Machine Learning Repository (provided by Prodigy InfoTech).

- Scikit-learn
- Matplotlib
- Google Colab

---

## 🔍 Project Workflow:

### 1. Data Loading:
- Imported dataset into Google Colab using Pandas

### 2. Data Preprocessing:
- Handled categorical variables using Label Encoding
- Checked for missing values and dataset structure

### 3. Feature Selection:
- Split dataset into input features (X) and target variable (y)

### 4. Train-Test Split:
- Data divided into training and testing sets (80/20 split)

### 5. Model Training:
- Applied Decision Tree Classifier using Scikit-learn

### 6. Prediction:
- Predicted customer purchase behavior (Yes/No)

### 7. Model Evaluation:
- Evaluated using accuracy score
- Generated classification report

### 8. Visualization:
- Visualized Decision Tree structure using Matplotlib

- Achieved good accuracy using Decision Tree Classifier.
